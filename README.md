# Laravel Design Tags Input

Tokenized tags input component for Laravel. Supports Livewire, Blade, and Vue 3.

## Installation

```bash
composer require mrshanebarron/tags-input
```


## Usage

### Livewire Component

```blade
<livewire:ld-tags-input />
```

### Blade Component

```blade
<x-ld-tags-input />
```

## Configuration

Publish the config file:

```bash
php artisan vendor:publish --tag=ld-tags-input-config
```

## Customization

### Publishing Views

```bash
php artisan vendor:publish --tag=ld-tags-input-views
```
